# Coder Commander

**Двухпанельный файловый менеджер для программиста**

![.NET](https://img.shields.io/badge/.NET-8.0-512BD4?logo=dotnet)
![WPF](https://img.shields.io/badge/WPF-Windows-0078D4?logo=windows)
![License](https://img.shields.io/badge/license-MIT-green)

## 🚀 Возможности

### 📁 Файловый менеджер
- **Две независимые панели** для эффективной работы с файлами
- Навигация по истории (вперёд/назад)
- Быстрый поиск и фильтрация файлов
- Операции: копирование, перемещение, удаление, переименование
- Поддержка скрытых файлов
- Закладки для быстрого доступа

### 💻 Встроенный терминал
- **CMD и PowerShell** в одном интерфейсе
- Нативная консоль Windows (не эмуляция)
- Выпадающая панель (SHIFT+F10)
- Автоматическая смена директории при навигации
- Поддержка UTF-8 и корректное экранирование путей
- Создание нескольких вкладок CMD/PowerShell

### 📝 Редактор кода
- Подсветка синтаксиса для 15+ языков:
  - C#, Java, C++, Python, JavaScript, TypeScript
  - HTML, CSS, XML, JSON, YAML
  - SQL, Bash, Dockerfile, Markdown, Git Diff
- Открытие в отдельных окнах с управлением (свернуть/развернуть/закрыть)
- Номера строк
- Поиск и замена
- Просмотр файлов (F3) и редактирование (F4)

### 🔧 Инструменты разработчика

#### Git
- Просмотр статуса репозитория
- Интерактивный staging/unstaging файлов
- Коммиты с визуальным diff
- Push/Pull/Fetch
- Управление ветками
- История коммитов
- Stash операции

#### Docker
- Список контейнеров и образов
- Запуск/остановка контейнеров
- Просмотр логов
- Docker Compose (up/down)
- Выполнение команд в контейнерах

#### SSH
- Управление SSH профилями
- Быстрое подключение к серверам
- Поддержка IdentityFile

#### SFTP
- Браузер удалённой файловой системы
- Загрузка/выгрузка файлов
- Создание папок, удаление, переименование

### ⚙️ Настройки
- **Темы**: Тёмная / Светлая / Как в системе
- **Редактор**: Выбор шрифта, размер, перенос строк
- **Терминал**: Выбор shell по умолчанию (CMD/PowerShell), шрифт
- **Поведение**: Подтверждения операций, автообновление

## 🛠️ Установка и запуск

### Из исходников

```bash
# Клонировать репозиторий
git clone https://github.com/romanborodin77/coder-commander.git
cd coder-commander

# Собрать проект
dotnet build

# Запустить
dotnet run
```

### Сборка Release

```bash
dotnet publish -c Release -r win-x64 --self-contained false
```

Исполняемый файл будет в `bin\Release\net8.0-windows\win-x64\publish\CoderCommander.exe`

## 🏗️ Архитектура

- **MVVM** паттерн с `CommunityToolkit.Mvvm`
- **WPF** для UI
- **AvalonEdit** для редактора с подсветкой синтаксиса
- **SSH.NET** для SSH/SFTP
- **Нативная Windows консоль** через Win32 API для терминала

### Структура проекта

```
CoderCommander/
├── Models/          # Модели данных
├── ViewModels/      # MVVM ViewModels
├── Views/           # XAML views
├── Services/        # Бизнес-логика
├── Converters/      # Value converters
└── Resources/       # Ресурсы и темы
```

## 🌟 Особенности

- ✅ **Нативный терминал** - не эмуляция, настоящий cmd.exe/powershell.exe
- ✅ **Два режима** - CMD и PowerShell в одном интерфейсе
- ✅ **Умная навигация** - терминал автоматически следует за активной панелью
- ✅ **Правильное экранирование** - корректная работа с путями, содержащими спецсимволы
- ✅ **UTF-8 поддержка** - правильное отображение кириллицы
- ✅ **Открытие в popup** - редактор и просмотр не занимают место в основном окне
- ✅ **Тёмная/Светлая/Системная тема** - комфортная работа в любое время суток
- ✅ **Стильные диалоги** - все MessageBox заменены на стильзованные окна с иконками
- ✅ **Окно «О программе»** - с анимированным SVG-логотипом `{ }` и интерактивом
- ✅ **Иконка приложения** - SVG-логотип `{ }` (6 размеров, ICO)
